# Task Manager

A simple task manager built with React and TypeScript.

## Stack

- React
- TypeScript
- Vite
- CSS
- Fake API layer with async/await
- LocalStorage

## Features

- View tasks list
- Create a task
- Edit a task
- Delete a task
- Search tasks
- Filter tasks by status
- Sort tasks by title
- Loading, error, empty, and success states
- Retry button on failed request
- Form submit protection while request is in progress

## Run locally

1. Install dependencies:
   npm install

2. Start the development server:
   npm run dev

3. Open the app in the browser using the local URL from Vite
